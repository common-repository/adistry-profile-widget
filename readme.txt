== Description ==

Easily display your Adistry Profile anywhere on your Wordpress website using shortcodes.

[adistry-profile-widget token="YOUR_TOKEN_HERE"]

Notes:

*   Each profile widget you create has it's own token.  Access this TOKEN from the "Partner Program" page on your channel's dashboard.
*   You may use the shortcode in any Blog or Page content, as well as the Text Widget (or anywhere else you set shortcodes to work)

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `adistry-profile-widget/` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the above shortcode to any Blog, Page, or Text Widget
